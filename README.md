# url-parse

[![npm version](https://img.shields.io/npm/v/url-parse.svg)](https://www.npmjs.com/package/url-parse)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Lightweight URL parser and builder (works in Node + browser)

## Installation

```bash
npm install url-parse
```

## Functions

- `parseUrl` — Parse URL into components
- `buildQuery` — Build query string from object
- `parseQuery` — Parse query string to object
